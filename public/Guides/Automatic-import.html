    Importing items from other sources

    We highly recommend setting up the automatic search functionality of Pure to easily import your research output. In this guide recommendations are given on how to set up automatic import for publications, including step-by-step explanations.

    Pure has the ability to import research output and datasets from the following databases:

    Which service should I use?

    Scopus. Explanations and details can be read by expanding the items below (click them).

    Do not enable more than 1 automatic import service!

    If you enable multiple automatic import services you will run into issues. Because the different databases will supply their data in different ways it‚Äôs difficult for Pure to immediately detect duplicate items. Although you‚Äôre always able to choose which data to add and fix any errors in metadata, it‚Äôs easy to make a mistake and pollute the database. If you want to know more details about the various import sources, read the text below.

    Why is Scopus recommended?

    Scopus will include the most complete and correct metadata of the available options. This is partly because Scopus and Pure are both Elsevier products, which enables them to fully integrate both services. The second best option to use is Crossref: most published academic work will have a DOI registered by Crossref. However, this often won‚Äôt include as much metadata as an import through Scopus, so you‚Äôll have to add any missing data afterwards.

    Importing from preprint and patent databases

    arXiv and Espacenet are databases with a more narrow scope; the former exclusively hosts preprints, and the latter is a patents database. Both patents and preprints are also indexed by Scopus, so it might feel redundant to use these. However, arXiv and Espacenet will probably have better coverage compared to Scopus, depending on various factors. The available metadata will also be different, so they can be a valuable source for more information.

    I recently started at the UT. Can I import research that I've done at other universities into Pure?

    Yes, you can. Yes, you can. Please see the QRCs on the Service Portal for more details. If you have an up-to-date ORCID profile, an easy way is to use the ORCID automatic import option in Pure to transfer over your data.

    Pay attention to the item category when importing from arXiv

    When you import a preprint from arXiv, it will be registered in our RIS as a working paper, which isn‚Äôt monitored as part of your main research output. If this preprint is import to your output make sure to change the category to a more appropriate one. See the preprint manual on this site for more information.
